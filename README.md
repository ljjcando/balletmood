# Ballet Mood

> 记录每一次优雅的进步

一款专为芭蕾舞者设计的微信小程序，采用莫兰迪燕麦色系，帮助舞者记录每日练习感悟、课程进展，追踪芭蕾术语学习。

---

## 核心功能

### 1. 日历记录
- 月视图日历，直观查看练习记录
- 点击已有记录进入**展示模式**（明信片卡片）
- 点击空白日期进入**编辑模式**（新建记录）
- 有记录的日期显示特殊标记，有图片的日期显示缩略图

### 2. 术语标签
- 内置常用芭蕾术语（Rond de jambe、Adagio、jete 等）
- 支持多选术语标签
- 支持添加自定义术语
- 长按术语可删除

### 3. 多课程展示
- 单日可记录多节课程
- 支持选择课程类型（入门、初级、中级、PBT 等）
- 记录每节课的时长
- 详情页展示所有课程标签，横向排列自动换行

### 4. 图片上传
- 支持从相册或拍照上传练习照片
- 自动裁剪为 3:4 竖向比例
- 详情页有图时显示大图，无图时隐藏图片区域

### 5. 数据统计
- 本月目标设定
- 本地存储，数据永久保存
- 支持编辑和删除历史记录

### 6. 身体部位标记
- 标记练习涉及的身体部位（核心、脚背、外开、背部、手臂）
- 多选胶囊标签形式

## 特别记录：本次优化重点 (v1.1.0)

### 1. 展示模式与编辑模式的分离
**问题**：原逻辑中，点击有图片的记录进入展示模式，无图片记录直接跳转编辑模式。

**优化**：
```javascript
// 统一入口逻辑：只要已有记录，均进入展示模式
if (record) {
  this.openPostcard(date);   // 展示模式
} else {
  this.openEditModal(date);   // 新建模式
}
```

**效果**：用户体验更连贯，可先查看再决定是否编辑。只有点击"重新编辑"按钮才进入编辑态。

---

### 2. 按需渲染的性能优化
**图片区域条件渲染**：
```xml
<view class="postcard-image-wrapper" wx:if="{{postcardData.photo}}">
```
- 无图片记录不渲染图片容器
- 文字内容自动上移，布局紧凑

**课程标签按需展示**：
```xml
<view class="postcard-courses">
  <text class="postcard-course-tag" wx:for="{{postcardData.courses}}" wx:key="index">{{item}}</text>
</view>
```
- 空数组时自动隐藏整个标签区域
- 使用 `filter` 过滤空值，确保只渲染有效数据

---

### 3. 表头排版精细化
| 项目 | 优化前 | 优化后 | 说明 |
|------|--------|--------|------|
| 顶部 padding | 50rpx | 30rpx | 收紧顶部，视觉重心上移 |
| 日期数字 | 100rpx | 96rpx | 约 48px，比例更协调 |
| 月份年份字号 | 26rpx | 24rpx | 更精致 |
| 星期字号 | 22rpx | 20rpx | 更精致 |
| 日期区域间距 | 无 | gap: 4rpx | 布局更紧凑 |

---

### 4. 课程标签位置重构
**优化前**：课程标签悬浮在日期下方，与品牌信息割裂

**优化后**：
```xml
<view class="postcard-badge">
  <text class="postcard-icon">🩰</text>
  <text class="postcard-location">Ballet Mood</text>
  <view class="postcard-courses">
    <text class="postcard-course-tag" wx:for="{{postcardData.courses}}">{{item}}</text>
  </view>
</view>
```

- 课程标签移至 "Ballet Mood" 后方，作为品牌后缀
- 形成完整的品牌展示区：`🩰 Ballet Mood [初级] [PBT]`
- 多标签横向排列，间距 14rpx，支持自动换行

## 使用说明

1. **新建记录**：点击日历上的空白日期
2. **查看记录**：点击有记录的日期，进入展示模式
3. **编辑记录**：在展示卡片中点击"重新编辑"
4. **删除记录**：展示模式或编辑模式中均可删除
5. **切换月份**：点击顶部左右箭头
